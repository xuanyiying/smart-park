# Smart Park Web管理端设计文档

## 项目概述

为智慧停车场系统构建Web管理后台，供物业管理员、运营方、系统管理员使用。

## 技术栈

| 技术 | 选择 | 理由 |
|------|------|------|
| 框架 | Next.js 16 | 与后端Kratos生态兼容 |
| 语言 | TypeScript | 类型安全 |
| UI | Tailwind CSS v4 + shadcn/ui | 现代简洁设计、高质量组件 |
| 状态 | Zustand | 轻量级状态管理 |
| 数据获取 | TanStack Query | React Query |

## 架构设计

```
web/
├── app/
│   ├── (auth)/            # 登录页
│   ├── (dashboard)/       # 管理后台
│   │   ├── layout.tsx     # 侧边栏+顶部导航
│   │   ├── page.tsx       # 首页/数据概览
│   │   ├── parking-lots/ # 停车场管理
│   │   ├── vehicles/     # 车辆管理
│   │   ├── orders/       # 订单管理
│   │   ├── billing/      # 计费规则
│   │   ├── devices/      # 设备控制
│   │   ├── users/        # 用户权限管理
│   │   └── reports/      # 报表统计
│   └── layout.tsx
├── components/
│   ├── ui/               # shadcn/ui基础组件
│   ├── layout/           # 布局组件
│   └── modules/          # 各模块业务组件
├── services/             # API调用层
├── lib/                  # 工具函数
└── types/                # TypeScript类型定义
```

## 功能模块

### 1. 数据概览 (Dashboard)
- 今日收入统计
- 今日进出场车辆数
- 在线设备数
- 7天趋势图表
- 快速操作入口

### 2. 停车场管理
- 停车场列表(分页、搜索)
- 新增/编辑停车场
- 停车场详情、状态管理

### 3. 车辆管理
- 车辆列表(分页、筛选)
- 月租车/临时车分类
- 绑定车主信息(姓名、电话)
- 月租有效期管理

### 4. 订单管理
- 订单列表(分页、状态筛选)
- 订单详情
- 支付状态筛选

### 5. 计费规则
- 规则列表
- 新增/编辑规则
- 规则类型(时段收费、优惠折扣)
- 启用/禁用

### 6. 设备控制
- 设备列表
- 在线状态监控
- 远程开闸
- 心跳日志

### 7. 用户权限管理
- 管理员列表
- 角色管理
- 权限配置

### 8. 报表统计
- 日报/月报
- 数据导出

## 视觉设计

- **主题**: 浅色主题、卡片式布局、充足留白
- **主色**: 蓝色系 (#3B82F6)
- **背景**: #F8FAFC
- **卡片**: 白色背景、圆角、轻微阴影
- **字体**: Inter/系统字体
